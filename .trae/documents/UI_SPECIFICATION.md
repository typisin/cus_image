# CusImage UI 设计规范

## 概述
本文档定义了 CusImage 图片编辑工具的 UI 设计规范，采用简约现代风格和蓝色科技主题配色方案。

## 1. 文本规范

### 1.1 字体选择
- **主要字体**: Inter (优先使用)
- **备选字体**: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif
- **字体风格**: 无衬线体，现代简洁，具有良好的可读性和专业性

### 1.2 字号层级
| 元素类型 | 字号 | 字重 | 行高 | 使用场景 |
|---------|------|------|------|----------|
| 主标题 | 2.5rem (40px) | 700 | 1.2 | 页面主标题 |
| 副标题 | 1.125rem (18px) | 400 | 1.5 | 页面副标题 |
| 品牌名称 | 1.875rem (30px) | 700 | 1.2 | 头部品牌标识 |
| 卡片标题 | 1.25rem (20px) | 600 | 1.4 | 功能卡片标题 |
| 正文描述 | 1rem (16px) | 400 | 1.6 | 卡片描述文本 |
| 辅助文本 | 0.875rem (14px) | 400 | 1.5 | 页脚、副标题等 |

### 1.3 字重规范
- **700 (Bold)**: 主标题、品牌名称
- **600 (SemiBold)**: 卡片标题、重要强调
- **400 (Regular)**: 正文、描述文本
- **300 (Light)**: 特殊装饰文本（可选）

### 1.4 行间距与字间距
- **行间距**: 1.5-1.6倍（根据字号调整）
- **字间距**: 默认（-0.2px微调优化）
- **段落间距**: 使用间距系统变量

## 2. 色彩规范

### 2.1 色彩体系
| 色彩角色 | 色值 | 使用比例 | 使用场景 |
|----------|------|----------|----------|
| 主色调 | #2563eb | 20% | 按钮、图标、重要元素 |
| 主色浅调 | #3b82f6 | 10% | 悬停状态、渐变 |
| 主色深调 | #1d4ed8 | 5% | 激活状态 |
| 白色 | #ffffff | 40% | 背景、卡片 |
| 灰色50 | #f9fafb | 15% | 页面背景 |
| 灰色200 | #e5e7eb | 5% | 边框、分隔线 |
| 灰色600 | #4b5563 | 3% | 次要文本 |
| 灰色800 | #1f2937 | 2% | 主要文本 |

### 2.2 功能色
| 状态 | 色值 | 使用场景 |
|------|------|----------|
| 成功 | #10b981 | 操作成功提示 |
| 警告 | #f59e0b | 警告信息 |
| 错误 | #ef4444 | 错误提示 |
| 信息 | #3b82f6 | 信息提示 |

### 2.3 对比度要求
- **正文文本**: ≥ 4.5:1 (WCAG AA)
- **大号文本**: ≥ 3:1 (WCAG AA)
- **重要信息**: ≥ 4.5:1

## 3. 间距规范

### 3.1 间距系统
| 尺寸 | rem | px | 使用场景 |
|------|-----|-----|----------|
| space-1 | 0.25rem | 4px | 微小间距 |
| space-2 | 0.5rem | 8px | 元素内边距 |
| space-3 | 0.75rem | 12px | 图标间距 |
| space-4 | 1rem | 16px | 标准间距 |
| space-5 | 1.25rem | 20px | 中等间距 |
| space-6 | 1.5rem | 24px | 卡片内边距 |
| space-8 | 2rem | 32px | 区块间距 |
| space-10 | 2.5rem | 40px | 大间距 |
| space-12 | 3rem | 48px | 章节间距 |
| space-16 | 4rem | 64px | 超大间距 |

### 3.2 元素间距
| 元素组合 | 间距值 | 备注 |
|----------|--------|------|
| 卡片之间 | space-8 (32px) | 网格布局间距 |
| 图标与文字 | space-6 (24px) | 垂直方向 |
| 标题与描述 | space-4 (16px) | 垂直方向 |
| 页面边距 | space-6 (24px) | 容器内边距 |

### 3.3 对齐方式
- **文本**: 左对齐（默认）
- **标题**: 居中对齐（主标题）
- **卡片内容**: 顶部对齐
- **网格布局**: 等分布局

## 4. 图标规范

### 4.1 图标风格
- **风格**: 线性图标，简约现代
- **描边**: 1.5px 统一描边
- **圆角**: 适度的圆角处理
- **一致性**: 保持视觉重量一致

### 4.2 图标尺寸
| 使用场景 | 尺寸 | 容器尺寸 |
|----------|------|----------|
| 卡片图标 | 48x48px | 64x64px |
| 品牌标识 | 24x24px | 48x48px |
| 操作图标 | 20x20px | 40x40px |

### 4.3 图标颜色
- **默认状态**: 主蓝色 (#2563eb)
- **悬停状态**: 主色浅调 (#3b82f6)
- **禁用状态**: 灰色400 (#9ca3af)

## 5. 卡片规范

### 5.1 卡片样式
- **背景色**: 白色 (#ffffff)
- **边框**: 1px 灰色200 (#e5e7eb)
- **圆角**: 1rem (16px)
- **阴影**: 轻度阴影 (shadow-sm)

### 5.2 卡片交互
- **悬停效果**: 上移4px，阴影增强
- **聚焦状态**: 蓝色轮廓指示
- **激活状态**: 边框颜色变深

## 6. 响应式设计

### 6.1 断点设置
| 设备类型 | 断点 | 布局调整 |
|----------|------|----------|
| 移动端小屏 | ≤ 480px | 单列布局，缩小间距 |
| 移动端 | ≤ 768px | 单列布局，调整字号 |
| 平板端 | 769px - 1024px | 双列布局 |
| 桌面端 | ≥ 1025px | 三列布局 |

### 6.2 移动端适配
- **字号调整**: 主标题缩小至 1.75rem
- **间距压缩**: 使用更紧凑的间距
- **布局优化**: 单列流式布局
- **触摸友好**: 增大点击区域

## 7. 动画与交互

### 7.1 动画时长
| 动画类型 | 时长 | 缓动函数 |
|----------|------|----------|
| 快速动画 | 150ms | ease |
| 标准动画 | 250ms | ease |
| 慢速动画 | 350ms | ease |

### 7.2 交互反馈
- **悬停效果**: 250ms 平滑过渡
- **点击反馈**: 150ms 快速响应
- **状态变化**: 明确的视觉反馈

## 8. 可访问性

### 8.1 键盘导航
- **Tab 顺序**: 逻辑顺序导航
- **焦点指示**: 清晰的焦点样式
- **跳过导航**: 提供跳过链接

### 8.2 屏幕阅读器
- **语义化标签**: 正确使用 HTML5 语义标签
- **ARIA 属性**: 适当的 ARIA 标签
- **替代文本**: 为图标提供描述文本

## 9. 浏览器兼容性

### 9.1 支持浏览器
- Chrome 88+
- Firefox 85+
- Safari 14+
- Edge 88+

### 9.2 特性降级
- **CSS 变量**: 提供回退值
- **Grid 布局**: 使用 Flexbox 降级
- **现代特性**: 渐进增强策略
