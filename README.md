# CusImage - 智能图片编辑工具

一个现代化的图片编辑工具，提供三种强大的AI功能：图片描述生成、智能抠图、字符艺术转换。

## 🚀 功能特性

- **Image Describer** - 为图片生成精准自然的文字描述，支持多语言输出
- **AI Cutout** - 智能抠图与背景移除，保留精细边缘与毛发细节  
- **Image to Unicode** - 将图像转为Unicode字符艺术，兼容终端与网页显示

## 🎨 设计系统

基于简约现代风格和蓝色科技主题，提供完整的UI设计规范：

- **色彩系统**: 蓝色主色调 (#2563eb) + 白色辅色调
- **字体系统**: Inter 字体家族，现代简洁
- **间距系统**: 8px基准的响应式间距体系
- **卡片设计**: 悬浮效果，优雅过渡动画
- **响应式**: 完美适配桌面、平板、手机

详细设计规范请参考 [UI_SPECIFICATION.md](UI_SPECIFICATION.md)

## 🎯 设计原则

1. **简约现代** - 清晰的视觉层次，专注核心功能
2. **用户友好** - 直观的交互设计，流畅的用户体验
3. **专业可靠** - 科技感的配色方案，建立用户信任
4. **响应式优先** - 移动端优先的设计理念
5. **性能优化** - 轻量级代码，快速加载
